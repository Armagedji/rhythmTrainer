# 🎵 Rhythm Trainer

Приложение для Android, помогающее развить чувство ритма через игровые механики.

## 📱 О проекте

Rhythm Trainer объединяет педагогические методики обучения ритму с элементами геймификации. Приложение создаёт метроном с заданным BPM, визуализирует движение нот и оценивает точность нажатий пользователя. Встроенная система званий и сохранение прогресса мотивируют продолжать тренировки.

## ✨ Возможности

- 🎯 Три уровня сложности с разным темпом (80, 100, 120 BPM)
- 👆 Оценка точности нажатий: Perfect (≤60 мс), Good (≤120 мс), Miss
- 🎨 Визуализация движения нот в реальном времени
- ⏸ Пауза с сохранением позиции нот
- 🏆 Система званий от Новичка до Легенды
- 📊 Сохранение лучших результатов для каждого уровня
- 🎛 Калибровка задержки ввода
- 🔊 Настройки звука и вибрации
- 📚 Раздел обучения с видеоматериалами

## 🛠 Технологии

| Слой | Технологии |
|------|------------|
| UI | Kotlin, Jetpack Compose, Material 3 |
| Аудио | C++, Oboe (низколатентный звук) |
| Связь | JNI (Java Native Interface) |
| Хранение | SharedPreferences |
| Видео | AndroidX Media3 (ExoPlayer) |

## 🏗 Архитектура

Приложение построено на трёхуровневой архитектуре:

- **UI Layer** (Kotlin / Jetpack Compose) — экраны, навигация, визуализация нот
- **Core Layer** (Kotlin) — логика уровней, система званий, настройки
- **Native Layer** (C++ / Oboe) — аудиодвижок, обработка нажатий, таймлайн

## 🎮 Как использовать

1. Выберите уровень на экране уровней
2. Нажмите «Запустить ритм» для воспроизведения метронома
3. Следите за нотами, движущимися к зелёной линии
4. Нажимайте «Нажми в ритм!» в момент пересечения линии
5. Получайте оценку точности и очки
6. По завершении уровня просматривайте результат
7. Накапливайте очки и открывайте новые звания

## 🚀 Сборка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Armagedji/RhythmTrainer.git
```
2. Откройте проект в Android Studio (требуется Hedgehog или новее)
3. Установите NDK для компиляции C++ кода
4. Запустите на устройстве с Android 8.0 (API 26) или выше
